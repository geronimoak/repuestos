PASO 1 — Subir archivos al repo GitHub (geronimoak/repuestos)
─────────────────────────────────────────────────────────────
Copiá los archivos de esta carpeta "deploy/" al repo local y hacé commit:

  git add .
  git commit -m "Deploy I-REP v3.0 - servidor Node.js"
  git push origin main

PASO 2 — Crear proyecto en Railway
───────────────────────────────────
1. railway.app/new → "Repositorio de GitHub"
2. Seleccioná: geronimoak/repuestos
3. Railway detecta Node.js automáticamente
